# 🏗️ 贾维斯助手APP - 项目架构总览

## 📐 架构设计原则

### Clean Architecture
```
┌─────────────────────────────────────────────────┐
│                   表示层                          │
│  (Presentation Layer)                            │
│  • Activity/Fragment                             │
│  • ViewModel                                     │
│  • Compose UI Components                         │
└─────────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────────┐
│                   领域层                          │
│  (Domain Layer)                                  │
│  • Use Cases                                     │
│  • Domain Models                                 │
│  • Repository Interfaces                        │
└─────────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────────┐
│                   数据层                          │
│  (Data Layer)                                    │
│  • Repository Implementations                   │
│  • Data Sources (API, Database, Cache)          │
│  • Manager Classes                               │
└─────────────────────────────────────────────────┘
```

## 🔄 数据流

### 用户交互流程
```
用户操作
   ↓
UI组件
   ↓
ViewModel (处理业务逻辑)
   ↓
UseCase (执行用例)
   ↓
Repository (获取数据)
   ↓
DataSource (API/Database)
   ↓
返回数据
```

### WebRTC通话流程
```
用户点击通话按钮
   ↓
CallScreen → MainViewModel
   ↓
WebRtcManager.initialize()
   ↓
WebRtcManager.createPeerConnection()
   ↓
WebRtcManager.createOffer()
   ↓
服务器信令交换
   ↓
WebRtcManager.setRemoteDescription()
   ↓
通话建立
```

### 语音识别流程
```
用户说话
   ↓
BaiduSpeechManager.startRecognition()
   ↓
AudioRecord开始录制
   ↓
音频数据 → 百度云API
   ↓
识别结果回调
   ↓
显示在UI上
```

### 文件传输流程
```
用户选择文件
   ↓
FileTransferManager.uploadFile()
   ↓
文件读取 → 分块上传
   ↓
进度回调 → UI更新
   ↓
上传完成 → 保存记录
```

## 🎯 技术栈

### 核心技术
- **Kotlin**: 100% Kotlin代码
- **Jetpack Compose**: 现代化UI框架
- **Coroutines**: 异步编程
- **Flow**: 响应式数据流

### 架构组件
- **MVVM**: Model-View-ViewModel架构
- **Clean Architecture**: 清晰的层次划分
- **Repository Pattern**: 仓库模式
- **Dependency Injection**: Hilt依赖注入

### 数据存储
- **Room**: 本地数据库
- **DataStore**: 键值对存储
- **SharedPreferences**: 简单配置

### 网络通信
- **WebRTC**: 实时音视频通信
- **Retrofit**: RESTful API
- **OkHttp**: HTTP客户端

### 第三方服务
- **百度云**: 语音识别和合成
- **WebRTC**: 音视频通话
