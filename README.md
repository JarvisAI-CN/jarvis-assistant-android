# 🚀 贾维斯助手APP - 项目README

## 📱 项目概述
贾维斯助手APP是一款现代化AI助手应用，具备通话、语音识别、任务管理、文件传输等核心功能，采用蓝色科技主题设计。

## 🎯 核心功能

### 📞 通话系统
- **WebRTC通话**：基于WebRTC的高质量音视频通话
- **智能重连**：网络不稳定时自动重连
- **音频处理**：回声消除、降噪、音量控制
- **通话质量监控**：延迟、丢包率、比特率实时监控

### 🔊 语音系统
- **百度云语音识别**：高准确率的语音转文字功能
- **语音合成**：文本转语音功能
- **流式识别**：实时语音识别，低延迟返回结果
- **多语言支持**：普通话、英语、粤语、四川话

### 📁 文件系统
- **文件上传/下载**：支持多种文件格式
- **传输进度监控**：实时显示传输进度
- **断点续传**：网络中断后恢复传输
- **传输历史记录**：记录所有文件传输

### 📝 任务系统
- **任务调度**：定时、重复任务
- **任务执行引擎**：并发控制、进度反馈
- **任务管理界面**：卡片设计、状态图标
- **任务生命周期管理**：创建、执行、完成、取消

### 🔄 通知系统
- **4个通知渠道**：通话、消息、文件、任务
- **通知动态更新**：实时更新通知内容
- **通知点击处理**：点击通知跳转相关界面
- **广播接收器**：处理通知按钮点击事件

### 🌐 网络系统
- **网络状态监控**：实时监控网络状态
- **网络质量评估**：评估网络质量
- **自适应比特率**：根据网络质量动态调整
- **智能重试策略**：网络错误自动重试

## 🏗️ 架构设计

### 架构模式
- **Clean Architecture**：清晰的层次划分
- **MVVM模式**：响应式状态管理
- **Hilt依赖注入**：简化依赖管理
- **Room数据库**：本地数据存储
- **Coroutines/Flow**：异步处理和数据流

### 项目结构
```
app/src/main/java/com/assistant/voip/
├── core/                    # 核心组件
│   ├── App.kt              # 应用入口
│   ├── AppStateManager.kt  # 应用状态管理
│   ├── PermissionManager.kt# 权限管理
│   └── SpeechManager.kt    # 语音管理器
├── data/                    # 数据层
│   ├── rtc/               # WebRTC管理
│   ├── speech/            # 语音处理
│   ├── file/              # 文件管理
│   ├── task/              # 任务管理
│   ├── audio/             # 音频处理
│   └── network/           # 网络管理
├── domain/                  # 领域层
│   ├── model/             # 领域模型
│   ├── repository/        # 数据仓库
│   └── usecase/           # 业务用例
└── presentation/           # 表示层
    ├── screens/           # 界面组件
    ├── viewModel/         # 视图模型
    └── ui/                # UI组件
```

## 🎯 快速开始

### 构建项目
```bash
# 克隆项目
git clone <项目地址>

# 进入项目目录
cd voip_app_project

# 构建调试版本
./gradlew assembleDebug
```

### 运行项目
```bash
# 安装到设备
adb install app/build/outputs/apk/debug/app-debug.apk

# 启动应用
adb shell am start -n com.assistant.voip/.core.App
```
